## hw3：判斷質數
這題解題關鍵一直在於怎麼處理質數 2 這個數字，因為我一開始的解法是：
if(n % i !== 0) {       
  return true;     
}   
return false;
但是這樣的做法就會變成 2 會跳出迴圈直接變成 false ，所以有時候換個方式想其實就能想通了，不要一直用同一個方向去試。
